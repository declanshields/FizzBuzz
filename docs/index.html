Customisable FizzBuzz

Project Explanation

This website is dedicated to the Customisable FizzBuzz project that I created. This project is based off the typical programming challenge of FizzBuzz, but with a twist. All of the outputs and variables can be altered by the user, meaning that rather than just outputing "Fizz" or "Buzz" on multiples of 3 or 5 respectively, you can change the amount of factors used, the values of those factors, as well as the text outputted when the line is a multiple of that factor. It also allows you to choose a start and end point for the loop, so instead of going from 1 to 100, it can go from 9 to 47, 45 to 146, etc.
