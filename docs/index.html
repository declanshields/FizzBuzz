Customisable FizzBuzz

Project Explanation

This website is dedicated to the Customisable FizzBuzz project that I created. This project is based off the typical programming challenge of FizzBuzz, but with a twist. All of the outputs and variables can be altered by the user, meaning that rather than just outputing "Fizz" or "Buzz" on multiples of 3 or 5 respectively, you can change the amount of factors used, the values of those factors, as well as the text outputted when the line is a multiple of that factor. It also allows you to choose a start and end point for the loop, so instead of going from 1 to 100, it can go from 9 to 47, 45 to 146, etc.

Demonstration of the project.

All of the code for the FizzBuzz project is stored in one header file, meaning that to implement it into a project, you only need to import the one file and make a call to the FizzBuzz function to execute it. I came up for the idea for this project when watching YouTube and seeing videos about this problem as an interview question. I then decided to create a fully Customisable version of the code so that it can be changed to the users specifications. In the header file the max factors is set to 10 by default but can easily be altered by changing that one value, alongside this are the values for the pauses inbetween inputs and outputs, meaning that it can be changed to output all the text at once rather than wait between each line.
